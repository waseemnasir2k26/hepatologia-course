# Hepatologia Course

**Plataforma de Educacion Medica — Dr. Julio Santiago Marcelo**

Medical education course platform for a Peruvian hepatologist specializing in cirrhosis patient care. Bilingual Spanish interface with student portal, video lessons, and payment integration.

## Features

- **3 Core Modules + 3 Bonuses** — Structured cirrhosis education program
- **Student Portal** — Video lessons via Vimeo Pro, progress tracking, downloadable resources
- **Quiz System** — Knowledge assessment with scoring
- **Real Patient Cases** — "Casos Reales" section with anonymized case studies
- **How It Works Page** — Visual system overview at `/como-funciona.html`
- **Mercado Pago Checkout** — Peru-native payment gateway (ready for integration)
- **6-Month Access** — Time-limited enrollment with auto-expiry
- **Medical Green Palette** — Professional healthcare design language
- **Fully Spanish** — All copy, UI, and CTAs in native Spanish
- **WordPress-Ready** — Designed for migration to WordPress + Tutor LMS
